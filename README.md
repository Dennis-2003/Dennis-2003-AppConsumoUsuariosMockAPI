# UsuariosApi Android App 📱

Aplicación de Android moderna desarrollada en **Kotlin** con **Jetpack Compose** que consume una API REST para visualizar información de usuarios.

## 🚀 Características
- Consumo de API REST con **Retrofit**.
- Interfaz reactiva con **Jetpack Compose**.
- Manejo de estados (Cargando, Error, Éxito).
- Imágenes remotas cargadas con **Coil**.
- Arquitectura limpia y documentada.

## 🛠️ Tecnologías utilizadas
- **Kotlin**: Lenguaje de programación.
- **Jetpack Compose**: UI declarativa.
- **Retrofit 2**: Cliente HTTP para Android.
- **GSON**: Convertidor de JSON a objetos Kotlin.
- **Corrutinas**: Programación asíncrona.
- **MockAPI**: Servidor de datos de prueba.

## 📦 Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/UsuariosApi.git
   ```
2. Abre el proyecto en **Android Studio**.
3. Sincroniza con Gradle y ejecuta en un emulador o dispositivo real.
